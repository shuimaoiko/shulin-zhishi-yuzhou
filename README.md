## 🚀 怎么看

**双击打开 [`index.html`](index.html)**（浏览器，完全离线）。页面默认就是「思维导图」：

| 视图 | 说明 |
|------|------|
| 🧠 **思维导图** | 树状图。根「树林知识宇宙」分出知乎 / 公众号 / B站动态三枝。公众号可切换「类型+时间 / 类型 / 不分来源」等组织方式；类型+时间会先分内容类型，再分公众号账号，再按日期排列文章。内容超过阈值会自动分成第1组、第2组，避免一次展开过多标题。 |

左侧 **系统 · 分组** 面板：点系统标题可整组开/关，点分组单独筛选；顶部搜索框按标题、正文摘录、账号和分类元数据过滤。顶部 **索引** 会打开独立弹窗：可以输入任意关键词查看导图或导出 Markdown，也可以在可滚动索引词列表里直接查看/导出已有索引词。搜索/关键词结果会先显示系统、类型、账号和分页组，点分页组才展开具体内容，避免一次铺开几百条。
点任意内容 → 右侧抽屉显示系统、分组、点赞/评论或日期、正文摘要、关联维度、原文链接（知乎或微信公众号）。
底部提示语仅在进入页面时短暂出现；右下角 `?` 打开精简帮助中心，`⟲` 重置当前视图布局。思维导图支持“无限画布式”拖动：拖拽空白处或文字标签区即可平移整张画布。

### Agent 调用

其他 Agent 可直接读取机器索引，不需要用户手动导出：

- `agent.html?q=内耗&platform=all&format=md`：按关键词返回 Markdown。
- `agent.html?q=内耗&platform=mp&format=json`：按关键词和平台返回 JSON。
- `agent.html?id=bili-1060&format=md`：按单条 `Agent ID` 返回具体内容。
- `数据/agent-index.json`：全量机器可读索引，包含 `search_text`、分类文档路径、原文链接和正文摘录。
- `数据/agent-index.js`：同一份索引的离线注入版，双击打开 `agent.html` 时也能读取，避免浏览器拦截本地 JSON fetch。

## 🔧 重建

```
cd /Users/水猫/文档/树林/树林知识宇宙
python3 _build/clean_sources.py   # 可选：重新执行公众号源文档清洗
python3 _build/parse.py     # 一并重生成 数据/data.json + 数据/data.js + 数据/agent-index.json/js + 数据/分类文档 + stats.txt
```

如需重抓/重生成 B站视频文案：先运行 `../b站视频/_build_crawl.py` 和 `../b站视频/_build_md.py`，再运行本项目的 `_build/parse.py`。

数据模型：`systems`(知乎/公众号/B站动态/B站视频) → `groups`(知乎13维度 + 公众号2号 + B站动态类型 + B站视频主题，各带 `sys`) → `items`(带 `sys` + `group`；公众号另带 `mp_type`)。
知乎维度关键词表在 `parse.py` 的 `CATS`；公众号账号配置在 `MP_ACCOUNTS`，公众号短分类配置在 `MP_TYPES`；B站动态类型配置在 `BILI_GROUPS`；B站视频主题配置在 `BV_CATS`。
